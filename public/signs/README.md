# Imagens dos sinais (LGP)

Coloca aqui uma imagem por letra do alfabeto manual da Língua Gestual
Portuguesa. O guia ("Ver alfabeto") mostra-as automaticamente.

## Como nomear

Um ficheiro **PNG** por letra, com o nome em **maiúscula**:

```
public/signs/A.png
public/signs/B.png
public/signs/C.png
public/signs/D.png
public/signs/F.png
public/signs/I.png
public/signs/L.png
public/signs/O.png
public/signs/U.png
public/signs/V.png
public/signs/W.png
public/signs/Y.png
```

## Recomendações

- Imagens quadradas (ex.: 256×256), fundo simples, mão bem visível.
- Usa referências **oficiais da LGP** — o alfabeto manual português é
  diferente de outros (ex.: o americano). Sinais corretos são essenciais
  numa app de inclusão.
- Enquanto uma letra não tiver imagem, o guia mostra a descrição escrita;
  nunca fica partido.
